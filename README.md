# Post-it game

Play the classic Post-it game on mobile with your friends at party or with discord !

> Made with **Node.js / Express.js / Socket.io / Vue.js / Vite js / Less**

## Process

**Backend Prod with Docker**

```
docker compose up -d
```

**Frontend**

```
npm install
```

```
npm run build
```

## Env vars

**Front**

- VITE_WEBSOCKET_URL: string

**Back dev**

- PORT: int
- FRONT_URL: string
